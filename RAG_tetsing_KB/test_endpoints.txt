       * Action: Copy the question the AI gives you.

          {
            "session_id": "test_all_features",

            "mode": "evaluate",
            "user_answer": "Decorators are functions that modify the behavior of another function without changing its source code."

  Test 3: Context Engineering (Long-Term Memory)
  Goal: Prove the system "remembers" old chats by archiving them to RAG.

   1. Start a specific topic: Send this message first:
       * POST /chat -> {"session_id": "test_all_features", "message": "My favorite color is neon green.", "mode": "learn"}
   2. Fill the buffer: Send 10 more short messages (e.g., "Hello", "Okay", "Continue", etc.) to the same session. This triggers the "archiving" of that
      first message.
   3. Test the Memory:
       * Endpoint: POST /chat
       * JSON Payload:
          {
            "session_id": "test_all_features",
            "message": "What is my favorite color?",

          }
  Summary of Success
  If all three work, you have a system that:
   1. Reads your documents (RAG).
   2. Grades your knowledge (Evaluation).
